CSS

(англ. Cascading Style Sheets — Каскадні Таблиці Стилів)

- це мова для управління зовнішнім виглядом HTML-документа.

За допомогою CSS можна задавати параметри відображення будь-якого тега веб-сайту: ширину і висоту, відступи, колір і розмір шрифту, фон і так далі.

Структура CSS

CSS складається з селекторів і властивостей:

селектор: {
властивість 1: значення1;
властивість 2: значення2;
...
}

За допомогою селекторів можна сказати браузеру які саме елементи ми хочемо оформити. Властивості описують як саме ми хочемо оформити ці елементи. Тобто селектори - це снайперський приціл, а властивості - це кисть, рубанок, скальпель і перфоратор.

Існує величезна кількість CSS-властивостей, які впливають практично на всі аспекти відображення елементів.

Причому кожній властивості відповідає певний набір значень.

Перевага стилів полягає в тому, що ви можете швидко і гнучко змінювати зовнішній вигляд потрібних елементів.
