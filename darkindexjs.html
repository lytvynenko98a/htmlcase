JavaScript(JS)

— динамічна, об'єктно-орієнтованa мова програмування.

JavaScript спочатку створювався для того, щоб зробити web-сторінки «живими».

Програми на JavaScript - звичайний текст. Вони не вимагають якоїсь спеціальної підготовки.

У цьому плані JavaScript сильно відрізняється від іншої мови, яка називається Java.


У мови Javascript спочатку була інша назва – «LiveScript».

Тоді була дуже популярна мова Java, і маркетологи вирішили, що схожа назва зробить нову мову більш популярною.

Планувалося, що JavaScript буде таким собі «молодшим братом» Java.

Однак, історія розпорядилася по-своєму:

JavaScript сильно виріс, і зараз це абсолютно незалежна мова, зі своєю специфікацією, яка називається ECMAScript, і до Java не має ніякого відношення.


    Область застосування Javascript:
  • Веб-додатки;
  • Браузерні операційні системи;
  • Серверні додатки;
  • Мобільні додатки;
  • Віджети;
  • Прикладне програмне забезпечення;
  • Офісні додатки (Excel);
  • та інше.


    Щоб підключити Javascript в HTML-документ, потрібно:
  1. Вписати тег-<script>, </script>
  2. І зовнішній файл “script.js” (<script src=“script.js”>)

Наприклад:

<script>
alert ( 'I am man' );
</script>
