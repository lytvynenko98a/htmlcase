Основні теги HTML5

<!DOCTYPE html>; <html>; <header> <footer>; <meta charset="імя кодування"> - що це все і як правильно використвувати?

<!DOCTYPE html>

Кожен HTML-документ повинен починатися з декларації типу документа або «доктайпів».

Тип документа потрібен, щоб браузер міг визначити версію HTML і правильно відобразити сторінку.

<html>

Найпростіша HTML-сторінка складається як мінімум з трьох тегів.

Тег <html> - це контейнер, в якому знаходиться весь вміст сторінки, включаючи теги <head> і <body>.

Як правило, тег <html> йде в документі другим після доктайпів.

<head>

Тег <head> призначений для елементів, мета яких - допомогти браузеру в роботі з даними. Вміст цього тега не відображається безпосередньо.

<body>

Тег <body> призначений для змісту веб-сторінки (контенту), що відображається у вікні браузера.

<header> <footer>

<header> - шапка сайту або розділу;

<footer> - підвал сайту або розділу.

<title>

Заголовок сторінки - це той текст, який відображається в лівому верхньому куті браузера, а також у вкладках.

Щоб задати заголовок сторінки, потрібно використовувати тег <title> всередині тега <head>.

<head>

<title> Вивчаємо HTML</ title>

</head>

<meta charset="імя кодування">

Кодування HTML-сторінки потрібно вказувати для того, щоб веб-браузер міг правильно відображати текст на сторінці. Якщо браузер неправильно «вгадає» кодування, то замість тексту будуть відображатися ієрогліфи.

Щоб повідомити браузеру кодування HTML-сторінки, необхідно всередині тега <head> використовувати тег:

<meta charset = "ім'я кодування">

Найпоширеніше сучасне кодування - utf-8. Використовуйте його у всіх своїх проектах. <meta charset = "utf-8">

Ключові слова

Теги <meta>, називаються мета-тегами. Їх використовують всередині тега <head>.

Мета-теги зберігають корисну для браузерів і пошукових систем інформацію. Один з таких тегів - це опис ключових слів сторінки.

В атрибуті content через кому перераховуються найважливіші слова зі змісту сторінки. Раніше цей тег був дуже важливий для пошуковиків. Який стан справ зараз - великий секрет Гугла

<meta name = "keywords" content = "різні, ключові, слова">

Опис змісту сторінки

Ще один корисний для пошукових систем мета-тег - короткий опис сторінки.

В атрибуті content має бути короткий зміст або анотація сторінки. Він часто використовується пошуковиками при відображенні результатів пошуку.

<meta name = "description" content = "короткий опис">
