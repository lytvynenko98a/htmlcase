Масиви

Базові поняття. Що таке "масив"? "var myArray"?

Масиви використовуються для того, щоб зберегти декілька значень в одній змінній. Значення елементів масиву можуть бути різних типів.

Приклади

var myArray1 = [];

var myArray2 = [1, 2, 3, 4];

var myArray3 = [1, 'text', 3, 0, 'ghjghgj'];

var myArray4 = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ];

var myArray5 = [1, 2, , 4];

myArray2 [1] = 333; // другий елемент масиву = 333, тобто myArray2 = [1, 333, 3, 4];

myArray2 [7] = 5; // отримаєм myArray2 = [1, 333, 3, 4, undefined, undefined, undefined, 5];

console.log (myArray); // виведе в консоль масив myArray

console.log (myArray[2]); // виведе в консоль 3 елемент масиву myArray

console.dir (myArray); // також виведе в консоль масив myArray, але більш відформатовано

Методи масиву

length - довжина масиву

console.log (myArray.length); // виведе в консоль довжину масиву myArray

Змінні типу string - це теж масив

var myString ='text about me';

console.log (myString [3]); // x

console.log (myString.length) // 13 пробіли теж рахує

join () - перетворює елементи масиву у стрічку

var str = myArray2.join ();

console.log (str); // 1,2,3,4

Може при цьому розділяти елементи масиву заданими символами.

var str = myArray2.join ('bla');

console.log (str); // 1bla2bla3bla4

sort - сортує алементи масиву в алфавітному порядку

var myArray5 = ['Marta', 'Alina', 'Olga'];

myArray5.sort ();

console.log (myArray5); // ['Alina', 'Marta', 'Olga']

reverse - сортує елементи масиву у зворотньому порядку

var myArray5 = ['Marta', 'Alina', 'Olga'];

myArray5.reverse ();

console.log (myArray5); // ["Olga", "Alina", "Marta"]

slice () - витягає з миву потрібні еласементи

var myArray6 = ['Marta', 'Alina', 'Olga', 'Oksana', 'Ira'];

var myNewArray6=myArray6.slice (1, 3);

//вибере елемети масиву від другого до четвертого, невключаючи четвертий (елемент з індексом1, елемент з індексом2)

console.log (myNewArray6); // ["Alina", "Olga"]

splice ()- видаляє з масиву вказані елементи (та/або вставляє)

var myArray6 = ['Marta', 'Alina', 'Olga', 'Oksana', 'Ira'];

myArray6.splice (0, 2); //видаляє два елемети масиву, починаючи від першого (елемент з індексом0, елемент з індексом1)

console.log (myArray6); // ["Olga", "Oksana", "Ira"]

var myArray7 = ['Marta', 'Alina', 'Olga', 'Oksana', 'Ira'];

myArray7.splice (1, 3, 'Olena', 'Marija'); //видаляє три елемети масиву, починаючи від другого та додає замість них нові елементи

console.log (myArray7); // ["Marta", "Olena", "Marija", "Ira"]

    Додавання та видалення елементів массиву:

  • push () – додає елемент або кілька елементів в кінець масиву і повертає його нову довжину

  • pop () – видаляє елемент або кілька елементів з кінця масиву і повертає його нову довжину

  • unshift () – додає елемент або кілька елементів на початок масиву і повертає його нову довжину

  • shift () – видаляє елемент або кілька елементів з початку масиву і повертає його нову довжину
